如果遇到其他环境依赖问题，尝试使用指定版本的安装方式
pip install -r requirements.version.txt


正常使用脚本执行：
pytohon Honkai_Star_Rail.py

自动捡漏在部分地图会卡住，所以不推荐开启


如果遇到其他奇怪的问题无法解决，可以尝试使用虚拟环境：
#虚拟环境创建
python -m venv env
#激活虚拟环境
.\env\Scripts\Activate
#给虚拟环境安装依赖
pip install -r requirements.version.txt
#运行脚本
pytohon Honkai_Star_Rail.py
后续使用需要先激活虚拟环境再执行脚本
.\env\Scripts\Activate
pytohon Honkai_Star_Rail.py
